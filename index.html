Problem 1: DFA for Odd Length Strings

Language: L = {w ∈ {a, b}* | |w| is odd}

Formal Definition

M = (Q, Σ, δ, q₀, F) where:

  • Q = {q₀, q₁} (states)
  • Σ = {a, b} (alphabet)
  • q₀ = initial state
  • F = {q₁} (accepting states)
  • δ = transition function (see table below)
State Input 'a' Input 'b' Description
q₀ q₁ q₁ Even length (non-accepting)
q₁ q₀ q₀ Odd length (accepting)

Example Strings

✓ Accepted: "a" (length 1)
✓ Accepted: "aba" (length 3)
✓ Accepted: "bbbbb" (length 5)
✗ Rejected: "" (length 0)
✗ Rejected: "ab" (length 2)
✗ Rejected: "aaaa" (length 4)

Problem 2: DFA for Strings Starting with 'b'

Language: L = {w ∈ {a, b}* | w starts with b}

Formal Definition

M = (Q, Σ, δ, q₀, F) where:

  • Q = {q₀, q₁, q₂} (states)
  • Σ = {a, b} (alphabet)
  • q₀ = initial state
  • F = {q₁} (accepting states)
  • δ = transition function (see table below)
State Input 'a' Input 'b' Description
q₀ q₂ q₁ Initial state
q₁ q₁ q₁ Accepting state (started with b)
q₂ q₂ q₂ Rejecting state (started with a)

Example Strings

✓ Accepted: "b"
✓ Accepted: "ba"
✓ Accepted: "baaab"
✓ Accepted: "bbbb"
✗ Rejected: "" (empty)
✗ Rejected: "a"
✗ Rejected: "ab"
✗ Rejected: "aaab"
